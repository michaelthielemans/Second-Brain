Analog -> digital convertor

MCP3008 is a ADC
- quantization in amplitude : 10 bit resolution = 0 -> 1023 values
- quantization in time: sample frequency = 200ksps (KiloSamplesPerSeconde)
- Interface = SPI

**The Pi uses 4 digital pins to control the AD:**
- CS/ : chip select  
- D IN: data IN  
- CLK : clock  
- D OUT: data out
