
#linux 
## default user creation settings:


#### user and group config files:
> /etc/passwd
> /etc/shadow
> 	layout : username:

> /etc/group
> /etc/gshadow
#### Default config Files for 'useradd': 
> /etc/default/useradd
> /etc/logindefs

### username conventions:
- max 32 char
- can start with '_ '  'a-z'
- from second char '_'    ' - '  alfanumeric

### service accounts
UID from 1 - 499 or 1 - 999
